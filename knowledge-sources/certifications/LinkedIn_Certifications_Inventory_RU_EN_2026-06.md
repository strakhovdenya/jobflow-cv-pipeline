## 5. LinkedIn / CV safety notes

- Do not put all certificates into a compact two-page CV. It will look unfocused.
- Keep 3–5 certificates in the CV sidebar, selected by vacancy.
- The strongest general-purpose certificates are: System Design in Cloud, Node.js Testing and Code Quality, Node.js Security, FastAPI 2026, Introduction to MCP.
- The strongest AI-specific certificates are the three Anthropic certificates plus OpenAI API / React certificate.
- Redis certificates are useful only when Redis/caching matters.
- Old Prometheus certificates are acceptable on LinkedIn but usually not useful in a modern targeted CV.
- Certificates should support the profile, not define it. The main selling points remain commercial Node.js/TypeScript, Azure serverless, production e-commerce integrations, testing, production debugging, and PostgreSQL foundation.
