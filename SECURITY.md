# Security Policy

## Supported Versions

This project has no release branches — only `main` is supported. Security
fixes are applied to `main` only; there are no maintained older versions.

## Reporting a Vulnerability

Please report security vulnerabilities privately using
[GitHub Security Advisories](https://github.com/strakhovdenya/jobflow-cv-pipeline/security/advisories/new)
for this repository. This creates a private report visible only to the
repository owner and does not disclose the issue publicly.

Do not open a public GitHub issue for security vulnerabilities.

You can expect an initial response within a few days. Once a fix is
available, the advisory may be published and, where applicable, a CVE
requested.
